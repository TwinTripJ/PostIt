# 📄 프로젝트 소개

####  "POSTIT"은 사용자가 여행 및 일상 게시글을 작성하고 열람할 수 있도록 하는 웹 애플리케이션입니다.

<br/>

# ✈ 개발 동기

####  여행 후기를 쉽게 기록하고 열람할 수 있는 공간이 필요하다고 느꼈으며,
####  다양한 여행 경험을 한눈에 볼 수 있도록 웹 애플리케이션을 개발하게 되었습니다.
